# 🎯 AI Resume Analyzer

Analyze your resume against any job description using AI.

## 🛠️ Tech Stack
- Groq (LLaMA 3.3)
- Streamlit
- PyMuPDF
- Plotly

## 🚀 How to Run
pip install -r requirements.txt
streamlit run app/main.py

## 📸 Features
- Match Score
- Skill Gap Analysis
- Visual Dashboard
